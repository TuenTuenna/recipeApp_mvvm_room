# recipeApp_mvvm_room
simple android recipe application using mvvm architecture design pattern 

#### used api 
- food2fork recipe api

https://www.food2fork.com/about/api

#### used technique
- mvvm architecture design pattern (without databinding)
- room local database (for caching data)

#### used 3rd party library
- glide (for image processing)
